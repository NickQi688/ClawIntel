# 高密度信息图生成器

将复杂专业知识转化为**超高信息密度**的小红书爆款干货内容。支持多种视觉风格：莫兰迪手账、波普网格、复古档案、热敏小票等。

---

## 🎯 核心功能

- **6步工作流程**：从主题询问到内容生成的完整流程
- **7模块高密度**：每张图包含6-7个信息模块，拒绝空洞
- **多种视觉风格**：支持手绘、波普、档案、小票等多种风格
- **数据驱动**：每个模块包含具体数字、品牌、参数

---

## 📋 使用方法

```
帮我生成小红书爆款信息图
```

然后按提示提供：
1. 主题：你想制作的干货主题
2. 简短描述：核心要点或目标受众
3. 图片数量：希望生成多少张（3-10张）
4. 风格选择：莫兰迪手账 / 波普网格 / 复古档案 / 热敏小票 / 文件夹 / 坐标蓝图

## 📐 工作流程

### 步骤1：启动询问
```
📝 请提供以下信息：
1️⃣ 主题：你想要制作的干货主题是什么？
2️⃣ 简短描述：用1-2句话描述核心要点
3️⃣ 图片数量：希望生成多少张图片？（3-10张）
4️⃣ 风格：选择视觉风格（默认：莫兰迪手账）
```

### 步骤2-3：搜索素材与提炼价值
- 搜索相关高赞内容
- 提炼实用/稀缺/痛点价值
- 收集具体数据（价格、参数、百分比）

### 步骤4：智能拆分
将价值点拆分为6-7个核心模块：
- 模块1：品牌/选项类
- 模块2：数值阶梯类
- 模块3：场景对比类
- 模块4：识别技巧类
- 模块5：避坑提醒类
- 模块6：对比表格类
- 模块7：快速总结类

### 步骤5：生成内容
按照选择风格生成详细内容结构

### 步骤6：确认生图
生成精确的图片生成 Prompt

## 🎨 风格详细参数

### 风格1：莫兰迪手账（默认）

**配色方案：**
- 背景：温暖米黄 #F5F0E6
- 主色：柔和蓝绿 #7BA3A8
- 强调色：暖橙棕 #D4956A
- 线条：深炭棕 #4A4540

**视觉元素：**
- 和纸胶带装饰
- 手绘圆角标签
- 点线连接框
- 小房子、星星、云朵装饰

---

### 风格2：波普网格

**配色方案：**
- 背景：温暖复古米黄 #F5F0E6
- 平涂色：鲑鱼粉、天蓝、芥末黄、薄荷绿
- 对比色：纯黑 #000000 和纯白 #FFFFFF

**视觉元素：**
- 严格瑞士网格布局
- 统一粗黑描边
- 反差模块（黑底白字）
- 复古波普图标

---

### 风格3：复古档案

**配色方案：**
- 背景：牛皮纸棕、米白
- 主色：深黑、藏蓝
- 强调色：鲜绿、鲜红、米黄

**视觉元素：**
- 撕纸边缘效果
- 红色图钉
- 回形针
- 拍立得相框
- 打字机字体

---

### 风格4：热敏小票

**配色方案：**
- 边框：亮青色 #00AEEF 或芥末黄 #FFD100
- 核心：米白/浅灰 #F9F9F9
- 文字：深炭黑
- 强调：与边框同色

**视觉元素：**
- 3D标签打印头效果
- 打孔边缘
- 复古像素字体（标题）
- 3D/粘土图标
- 手绘高亮标记

---

### 风格5：文件夹

**配色方案：**
- 背景：米黄 #F5F5DC
- 主色：克莱因蓝
- 强调色：鲜艳橙
- 文字：浅灰

**视觉元素：**
- 竖式剪贴板
- 分层文件夹
- 索引标签
- 3D鼠标光标
- 通知图标

---

### 风格6：坐标蓝图

**配色方案：**
- 背景：专业灰白或蓝图网格 #F2F2F2
- 系统基色：柔和蓝绿 #B8D8BE
- 高警示：荧光粉 #E91E63
- 标记高亮：柠檬黄 #FFF200
- 线条：超细炭棕 #2D2926

**视觉元素：**
- 坐标标注（R-20, G-02, SEC-08）
- 技术图纸（爆炸图、截面图）
- 精密刻度尺
- 数学符号（Σ, Δ, ∞）
- 条形码、时间戳

## 💡 使用技巧

1. **选择合适风格**：生活类选手账，技术类选蓝图，潮流类选波普
2. **数据要具体**：价格、百分比、品牌名
3. **模块要丰富**：宁可信息丰富，不可内容空洞
4. **字体可缩小**：优先信息密度，而非字体大小
