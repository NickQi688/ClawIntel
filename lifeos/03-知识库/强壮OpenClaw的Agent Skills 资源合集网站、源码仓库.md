本文整理了当前最值得关注的 Agent Skills 合集网站和源码仓库，帮助你快速定位所需资源。随着 AI Agent 技术的快速发展，技能（Skills）已成为提升 Agent 能力的核心要素。无论是想快速为自己的 Agent 添加新功能，还是深入研究技能的工程实现，都需要知道去哪里找到高质量的资源。

## 一、合集网站：快速使用与发现

这类平台的核心价值在于“即用性”——你可以直接浏览、搜索并安装技能，无需关心底层实现细节。

1\. [agent-skills.md](http://agent-skills.md/)：高频技能的首选

**网站地址**：[https://agent-skills.md/](https://agent-skills.md/)

**核心特点**：收录超过 6000+ 常用、高频技能，强调“开箱即用”。

**适用场景**：当你需要快速为 Agent 添加某个常见功能时，这里往往能找到现成的解决方案。比如需要天气查询、日程管理、数据转换等标准功能，这里的技能库覆盖面广，上手快。

**推荐理由**：数量庞大且更新活跃，适合作为技能搜索的第一站。

2\. Skills Directory：社区口碑榜单

**网站地址**：[https://www.skillsdirectory.com/](https://www.skillsdirectory.com/)

**核心特点**：Reddit 社区推荐与整理，更像是一个“口碑榜单”。

**适用场景**：当你不确定某个技能是否好用时，可以先来这里看看社区评价。用户反馈和讨论能帮助你避开质量不佳的技能，找到真正经过实战检验的方案。

**推荐理由**：社区驱动的筛选机制，质量相对有保障。

3\. SkillsMP：全网技能聚合器

**网站地址**：[https://skillsmp.com/zh](https://skillsmp.com/zh)

**核心特点**：聚合 GitHub 上超过 11 万个开源技能，支持溯源到原始仓库。

**适用场景**：当你需要“全网搜技能”，或者想找到某个技能的源码进行定制时，这里是最全面的索引平台。特别适合开发者深度使用。

**推荐理由**：覆盖面最广，且能直接跳转到 GitHub 查看源码和文档。

4\. Agent Skills Me：精选少而精

**网站地址**：[https://agentskills.me/](https://agentskills.me/)

**核心特点**：人工精选，强调“精而少”。

**适用场景**：当你不想花时间在海量技能中筛选，希望有人帮你做好第一轮过滤时，这里提供的都是经过人工审核的高质量技能。

**推荐理由**：适合快速决策，节省选择成本。

5\. SkillStore：中文友好的安全选择

**网站地址**：[https://skillstore.io/zh-hans](https://skillstore.io/zh-hans)

**核心特点**：中文界面友好，强调技能经过安全审查。

**适用场景**：特别适合中文用户和团队使用，尤其是在合规性要求较高的场景下（如企业环境），安全审查机制能降低风险。

**推荐理由**：本土化支持好，安全性有保障。

6\. [Skills.sh](http://skills.sh/)：热门趋势发现平台

**网站地址**：[https://skills.sh/](https://skills.sh/)

**核心特点**：偏向热门趋势技能发现，支持一键安装。

**适用场景**：当你想尝试最新、最热门的技能时，这里能帮你快速发现和体验新功能。适合喜欢尝鲜的用户。

**推荐理由**：紧跟技术趋势，安装体验流畅。

## 三、如何选择适合自己的资源？

面对如此多的平台和仓库，如何快速找到适合自己的资源？以下是一些实用建议：

**如果你是普通用户**，只想快速为 Agent 添加功能：

- 首选 [agent-skills.md](http://agent-skills.md/) 或 **SkillStore**，数量多且易用
- 如果在意口碑，可以先去 **Skills Directory** 看看评价

**如果你是开发者**，需要定制或深度集成：

- 先去 **SkillsMP** 搜索，找到源码仓库
- 参考 **Anthropic Skills** 了解官方标准
- 学习 **Antfu Skills** 提升工程化水平

**如果你在企业或团队环境**：

- 优先考虑 **SkillStore**，安全审查机制更放心
- 参考 **Vercel Agent Skills** 了解生产环境集成方案

**如果你喜欢尝鲜**：

- 关注 [Skills.sh](http://skills.sh/)，第一时间体验热门新技能

从这些平台和仓库的发展来看，Agent Skills 生态正在呈现几个明显趋势：

2. **安全性**：安全审查和合规性成为重要考量，尤其在企业场景
3. **社区驱动**：开源社区的力量正在塑造技能生态的发展方向
4. **专业化分工**：从“大而全”到“精而专”，细分领域的专业技能库开始涌现
