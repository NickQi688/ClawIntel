# 决策逻辑：AI视频项目评估模型 (v1.0)

## 1. 核心评估维度 (Score 1-10)
- **ROI (回报率)**: (预期收益 - 时间成本) / 难度。
- **SOP 兼容度**: 是否能复用现有的 Seedance/Remotion 工作流。
- **正反馈潜力**: 甲方是否愿意公开合作案例（品牌价值）。

## 2. 决策逻辑
- **评分 > 24**: 💡 **立刻行动**。直接进入 Phase 1 需求确认，无需反复讨论。
- **18 < 评分 <= 24**: 💬 **有条件接单**。要求预付比例提升或样片通过后再批量。
- **评分 <= 18**: ⛔ **拒绝/转包**。除非有极高的战略意义（如建立新行业标杆）。

## 3. 拦截触发词
- "先试做几天" -> **警告**: 拒绝免费试做，建议提供现有作品集。
- "工期非常紧，明天要" -> **拦截**: 除非有 50% 的加急费。

## 4. 复盘优化点
- 每次项目后，若 (实际工期 > 预估工期) 2 倍，则调高该类项目的难度权重。
