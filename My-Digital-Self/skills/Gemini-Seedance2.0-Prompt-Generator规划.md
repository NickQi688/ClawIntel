## 一、核心定位

**与 Seedance 2.0 的区别**：
- Seedance 2.0：视频生成模型（即梦出品）
- 本 Skill：**为 Seedance 2.0 生成精准提示词**，基于 Gemini 3.5 Flash

**核心价值**：
- 快速生成专业级 Seedance 2.0 提示词
- 3个版本供选择（基础/专业/创意）
- 支持对话优化调整
- 成本低（Gemini Flash 极具性价比）

## 三、10 大场景支持

基于 Seedance 2.0 手册，完整支持以下场景：

### 1. 纯文本生成视频
```markdown
**用户输入**：文字描述需求
**生成提示词**：场景描述 + 角色设定 + 画面风格
**示例**：
- 基础版：一个女孩在海边跑步，日出时分，温暖色调
- 专业版：电影级广角镜头，女孩在沙滩上慢跑，金色阳光穿透云层，海浪轻柔拍岸，色调温暖饱和，4K画质
- 创意版：第一人称视角奔跑镜头，海风吹动头发，脚步溅起水花，呼吸声与海浪声交织，情绪自由奔放
```

### 2. 一致性控制
```markdown
**功能**：保持角色/产品/场景统一
**支持**：
- 角色一致性（服装、发型、五官）
- 产品一致性（logo、包装、质感）
- 场景一致性（光影、色调、氛围）
```

### 3. 运镜和动作复刻
```markdown
**参考视频**：用户提供参考视频
**生成提示词**：
- 完全复刻运镜方式（推/拉/摇/移/环绕）
- 精准模仿动作节奏
- 匹配转场效果
**Gemini 优势**：100 万 tokens 可完整分析视频描述
```

### 4. 创意模板复刻
```markdown
**支持**：
- 广告成片风格
- 电影片段特效
- AI 视频转场效果
**示例**：参考某品牌广告，用你的产品重新创作
```

### 5. 剧情创作/补全
```markdown
**功能**：
- 生成完整剧情脚本
- 延续已有故事线
- 创建分镜脚本
**输出**：带时间戳的分镜描述 + 对话 + 音效说明
```

### 6. 视频延长
```markdown
**输入**：原视频 + 延长时间
**生成**：无缝衔接的延长部分提示词
**支持**：5s/10s/15s 延长
```

### 7. 声音控制
```markdown
**功能**：
- 音色参考（指定声音风格）
- 对白生成（AI 语音转文字）
- 音效设计（环境音、动作音）
**支持格式**：时间戳 + 音效描述
```

### 8. 一镜到底
```markdown
**定义**：无剪辑、连续镜头
**生成**：长镜头连贯性提示词
**应用**：沉浸式体验、纪录片风格
```

### 9. 视频编辑
```markdown
**功能**：
- 角色替换（更换人物）
- 剧情修改（颠覆原设定）
- 局部调整（增加/删除/修改）
**优势**：无需重头生成，定向修改
```

### 10. 音乐卡点
```markdown
**输入**：参考音乐 + 画面素材
**生成**：节奏匹配的镜头切换提示词
**特点**：画面与音乐精准同步
```

## 四、核心工作流

```
用户输入需求
    ↓
【阶段1】需求分析
  - 识别场景类型（10选1）
  - 分析参考素材（图/视频/音频）
  - 提取关键要素
    ↓
【阶段2】提示词生成（3版本）
  - 基础版：简洁可用
  - 专业版：专业术语 + 技术参数
  - 创意版：艺术化表达 + 创新视角
    ↓
【阶段3】对话优化
  - 用户选择版本
  - 提出修改意见
  - 迭代优化（可多轮）
    ↓
【阶段4】最终输出
  - 复制即用提示词
  - 参数建议（时长/比例/帧率）
  - 使用提示
```

## 六、核心提示词模板

### 系统提示词（System Prompt）
```markdown
你是 Seedance 2.0 提示词生成专家。你的任务：

1. **理解用户需求**：识别用户想要的视频类型（10 大场景之一）
2. **分析参考素材**：提取图片/视频/音频中的关键元素（运镜/动作/风格/节奏）
3. **生成专业提示词**：符合 Seedance 2.0 规范，包含：
   - 画面描述（清晰、具象）
   - 运镜语言（推/拉/摇/移/环绕/特写/全景）
   - 时间节点（0-3秒/3-8秒/8-12秒等）
   - 风格/色调/情绪
   - 音效/音乐建议
4. **输出3个版本**：
   - 基础版：简洁直接，快速生成
   - 专业版：电影术语，技术参数精确
   - 创意版：艺术化表达，创新视角

**重要约束**：
- 严格遵循 Seedance 2.0 参数限制（≤9张图、≤3个视频、≤3个音频、总时长≤15s）
- 不支持写实真人脸部（明确告知用户）
- 优先使用 @引用语法指定参考素材用途
- 提示词要具体可执行，避免模糊抽象

**输出格式**：
```markdown
## 基础版
[提示词内容]

## 专业版
[提示词内容]

## 创意版
[提示词内容]
```
```

## 七、API 调用示例

### Python 代码示例
```python
import google.generativeai as genai

# 配置 API
genai.configure(api_key="YOUR_API_KEY")
model = genai.GenerativeModel("gemini-2.5-flash")

# 生成提示词
prompt = f"""
用户需求：{user_request}
参考素材：
- 图片：{image_descriptions}
- 视频：{video_descriptions}
- 音频：{audio_descriptions}

请生成3个版本的 Seedance 2.0 提示词
"""

response = model.generate_content(prompt)
print(response.text)
```

### 首帧提示词优化工作流

**为什么要优化首帧提示词？**
- 首帧决定视频整体风格和质量
- Seedance 2.0 依赖首帧建立场景基调
- 高质量首帧提示词 → 更好的人物一致性、光影、构图

#### 协作流程

```
用户需求输入
    ↓
【阶段1】调用 Nano Banana Pro Skill
  - 识别首帧需求（人物/风景/产品等）
  - 触发关键词：头像、封面、社媒、缩略图、产品照
  - 获取 3 个推荐提示词 + 示例图片
    ↓
【阶段2】Seedance 2.0 提示词生成
  - 基于选中的 Nano Banana Pro 模板
  - 结合 Seedance 2.0 场景（10 大场景之一）
  - 转换为视频提示词格式（加入运镜、时间节点、音效）
    ↓
【阶段3】首帧生成
  - 使用优化后的首帧提示词生成视频
  - 保证首帧质量 = Nano Banana Pro 级别
    ↓
【阶段4】视频延续
  - 基于首帧风格，生成后续帧
  - 保持风格一致性
```

### 应用示例

### 调用时机

**应该调用 Nano Banana Pro Skill 的时候**：
- 用户需要视频封面/缩略图
- 用户需要人物首帧介绍
- 用户需要产品展示开头
- 用户不确定首帧风格，需要灵感
- 用户要求"高质量"、"专业级"首帧

**不需要调用的时候**：
- 用户已有明确首帧参考图
- 用户延续现有视频风格
- 用户首帧需求非常简单（如纯黑背景）

### 重要说明：灵活工作流

**不是一次性生成**：工作流支持多种路径，用户可根据需求选择：

#### 路径 A：完整流程（提示词库 → 图片 → 视频）
```
用户需求 → Nano Banana Pro 推荐 3 个提示词
    ↓
用户确认/优化提示词
    ↓
用户选择：生成首帧图片（Nano Banana Pro）
    ↓
用户返回：带首帧图片 + 视频提示词
    ↓
Seedance 2.0 生成视频
```

#### 路径 B：直接生成（已有图片/提示词）
```
用户需求 + 现有首帧图片/提示词
    ↓
Gemini Seedance Skill 直接生成视频提示词
    ↓
Seedance 2.0 生成视频
```

#### 路径 C：图片生成后视频
```
用户需求 → Nano Banana Pro 推荐 3 个提示词
    ↓
用户确认/优化提示词
    ↓
用户去其他地方生成首帧图片（Midjourney、Stable Diffusion 等）
    ↓
用户返回：带首帧图片 + 视频提示词
    ↓
Seedance 2.0 生成视频
```

#### 路径 D：纯提示词生成
```
用户已有首帧图片 + 明确提示词需求
    ↓
跳过 Nano Banana Pro
    ↓
Gemini Seedance Skill 直接基于需求 + 图片生成视频提示词
    ↓
Seedance 2.0 生成视频
```

### 首帧图片提示词：按需决定

**重要**：不是每个视频都需要首帧图片提示词，而是**根据用户需求决定**。

#### 决策流程
```
用户需求输入
    ↓
【步骤1】分析需求
  - 是否需要首帧图片？（参考：是否有明确的首帧风格要求）
  - 是否已有现成的图片/提示词？
    ↓
【步骤2】生成视频提示词方案
  - Gemini Seedance Skill 基于需求生成 3 个版本提示词
  - **基础版**：简洁直接，不包含首帧图片提示词
  - **专业版**：专业术语，不包含首帧图片提示词
  - **创意版**：艺术化表达，不包含首帧图片提示词
    ↓
【步骤3】用户确认方案
  - 用户选择版本（1/2/3）
  - 用户确认后，询问：
    "视频提示词已准备好，**是否需要首帧图片提示词**？"
    ↓
【步骤4A】需要首帧图片
  - 调用 Nano Banana Pro 生成首帧图片提示词
  - 追加到原视频提示词方案前
  - 生成完整方案：视频提示词 + 首帧图片提示词
    ↓
【步骤4B】不需要首帧图片
  - 直接输出原视频提示词方案
  - 用户可自行使用 Seedance 2.0 API 生成视频
```
