# 数字分身激活指南

## ✅ 已完成的配置

### Claude Code
已安装 `digital-self` 插件，配置了以下技能：
- `/boot` - 激活数字分身
- `/数字分身` - 激活数字分身
- `/激活数字分身` - 激活数字分身

**使用方法：**
打开 Claude Code 后，直接输入：
```
/boot
```
或
```
激活数字分身
```

### OpenCode
已创建 agent 配置文件：
- `/Users/a9999/.local/share/opencode/agents/boot.json`
- `/Users/a9999/.local/share/opencode/agents/数字分身.json`

**使用方法：**
1. 方式1：使用 `/boot` 命令
2. 方式2：说"切换到数字分身 agent"

## 🚀 最简单的方式（推荐）

### 在 Claude Code 中：
直接说：`激活数字分身` 或 `boot`

### 在 OpenCode 中：
直接说：`切换到 boot agent`

## 🔄 如果不工作

### Claude Code
```bash
# 重新加载配置
source ~/.zshrc
claude
```

### OpenCode
```bash
# 手动指定 agent
opencode --agent boot
```

## 📝 备注
所有配置都使用绝对路径，在任何目录下都能正常工作。
