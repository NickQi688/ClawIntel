## 03\. 视频生成组件开发

本章将帮助你配置网站的视频生成功能。我们已经为你准备好了一个多模型视频组件，你只需要根据自己的需求进行简单调整即可。完成本章后，你的网站将具备完整的视频生成能力，用户可以通过输入文字或上传图片来生成视频。

首页的视频组件是吸引用户的核心功能。这个组件支持多种 AI 模型，用户可以选择不同的模型来生成视频。组件的基本界面如下图所示。

![](https://sphere-sh.oss-cn-shanghai.aliyuncs.com/private/docx/drive/media/KmW4b7IsaohEVtxxEmNcgwTWnTh_hhkc?x-oss-credential=LTAI5tDANFnEjK2qWnHwZKzr%2F20260317%2Fcn-shanghai%2Foss%2Faliyun_v4_request&x-oss-date=20260317T013345Z&x-oss-expires=86400&x-oss-signature=65d8a0229e438e8b9573df5e518ee94123c2e1f07cbfe34849f242fb2769c300&x-oss-signature-version=OSS4-HMAC-SHA256)

组件默认提供三种不同的 AI 模型供用户选择。如果你只想做单模型站点，可以去掉模型选择栏，让界面更简洁。

这里有一个优化建议。如果你选择单模型模式，就不需要再区分文字生成视频和图片生成视频两种模式了。因为用户不上传图片时，系统会自动按文字生成视频处理，上传图片后则自动切换为图片生成视频。这样可以减少用户的选择负担，提升使用体验。

![](https://sphere-sh.oss-cn-shanghai.aliyuncs.com/private/docx/drive/media/KMhlbzDnYohPrPxSgMAciuK4nod_hhkc?x-oss-credential=LTAI5tDANFnEjK2qWnHwZKzr%2F20260317%2Fcn-shanghai%2Foss%2Faliyun_v4_request&x-oss-date=20260317T013345Z&x-oss-expires=86400&x-oss-signature=12fcf74fb6a46c496fae9fd1faf6643f8f9be17575b8b31d6e0259ca28c54187&x-oss-signature-version=OSS4-HMAC-SHA256)

模板已经为你预置了 Evolink 平台的四款主流模型 API。你只需要填入 API Key 就能直接使用。如果想接入其他模型厂商的 API，可以参考模板文档进行配置。

![](https://sphere-sh.oss-cn-shanghai.aliyuncs.com/private/docx/drive/media/QeIpbwrayovSzXxkwIBcgssanoP_hhkc?x-oss-credential=LTAI5tDANFnEjK2qWnHwZKzr%2F20260317%2Fcn-shanghai%2Foss%2Faliyun_v4_request&x-oss-date=20260317T013345Z&x-oss-expires=86400&x-oss-signature=655c2029d1a6bc69dfe7a47041822ee79024004753c80bf079864b03ad4f5559&x-oss-signature-version=OSS4-HMAC-SHA256)

这部分配置需要根据你选择的模型厂商能力来调整。目前组件已经针对 Evolink 的四种模型做好了参数对应关系。如果你使用的正是这四种模型之一，修改工作量会很小。

https://cdn.gooo.ai/web-images/9a574a469abed638f82c5b3967c7db0be7558ea8a6da23902af9ae938cfb2103

访问 Evolink 官网进行注册。网址是 https://evolink.ai/zh

注册完成后，进入控制台的密钥管理页面。点击创建新密钥按钮，系统会为你生成一个 API Key。这个密钥就是你调用 AI 模型的凭证，请妥善保管。

Evolink 平台上有多个渠道提供相同的模型，为了避免混淆，这里列出当前四种模型在 Evolink 后台的准确对应关系。

Sora 2 模型的链接是 https://evolink.ai/zh/sora-2

Wan 2.6 模型的链接是 https://evolink.ai/zh/wan-2-6

Veo 3.1 模型使用的是 veo 3.1 fast lite 版本，链接是 https://evolink.ai/zh/veo-3-1

Seedance 1.5 Pro 模型的链接是 https://evolink.ai/zh/seedance-1-5-pro

接下来我们把 Evolink 的 API 放到.env.local 使AI生视频功能可以正常跑通。

工具页的视频生成组件功能与首页组件基本一致，但当前样式还需要你进行一些微调。后续我们会更新优化后的版本并同步给你。
