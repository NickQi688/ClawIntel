1\. Cursor 编辑器的下载与安装

Cursor 是目前最好用的 AI 代码编辑器之一，界面友好，上手简单。当然如果你熟悉使用 VS Code 也是可以的。

访问官网 https://cursor.com ，网站会自动识别你的系统（Windows 或 Mac），点击下载按钮获取安装包。

1.3 首次启动和账号注册

首次启动 Cursor 需要登录。推荐使用 GitHub 账号登录，也可以使用 Google 账号或邮箱注册。

登录后选择 Free Plan（免费版）即可，对于新手学习完全足够。

2.1 为了让大家更好地理解，我们先介绍三个核心概念：

Claude Code：这是我们使用的编程工具（外壳），它可以理解你的项目结构并编写代码。但使用官方服务较贵，一般20美金的用不了多久，更高级的服务就是100美金和200美金一个月了，所以下面我们会用国产模型智普的 GLM 来代替 Claude Code 的内核，但实际写代码能力不会弱太多，是一个经济实惠的组合。

GLM 4.7：这是智谱 AI 发布的国产开源大模型（大脑），我们将用它来驱动 Claude Code，替代昂贵的国外模型，实现稳定且免费的 AI 编程。

cc-switch：这是一个切换器，帮助我们将 Claude Code 的大脑从国外模型切换为国内的 GLM 模型。

Claude Code 有好几种使用方式，如果是程序员 习惯命令行界面的，可以使用命令行方式来 Vibe Coding。

我们这里教大家安装的是用 Cursor（或 VS Code）里面的 Claude Code 插件的方式，会打开一个对新人更加友好的对话框模式使用。

如果希望使用命令行的方式使用 Claude Code，安装方法可以参照 Claude Code 航海手册： https://scys.com/course/detail/98?chapterId=4235

第一次接触编程开发工具的圈友可能对这类工具的界面会有点懵，而且 Cursor 针对特殊场景有不同的适配面板，对于新人来说也会有点复杂，所以我们先从配置界面开始。

打开软件后先用其随意打开一个文件夹。

打开后可能是非常简单的，此时我们点击右上角的齿轮

可以先按照下图配置界面：

Editors：编辑主界面，最中间的大区域，在 Cursor 的逻辑里这是可以隐藏的，因为不懂代码的人 Vibe Coding 可以不看代码，但我们还是建议打开使用。

Panel：底部面板，常用的终端命令行在这里。

其他的 Agents 和 Chat 都是 Cursor 的服务，免费账户额度不多但能够救济用用。

先在在左上角选择扩展插件按钮，点击展开插件列表。在搜索栏搜索“Claude Code”，找到 Anthropic 官方插件，点击 Install 安装，期间需要信任。

安装好之后，可以从右上角省略号里面打开

此时我们还没有切换内核到 GLM 4.7 模型，所以要我们登录 Claude Code 账号，但我们不用管可以先直接关掉，接下来设置 cc-switch。

cc-switch 是一个专门用来切换 Claude Code 底层模型的工具。Cursor 默认使用国外模型，网络不稳定且昂贵。通过 cc-switch，我们可以一键切换到国产智谱 GLM 模型，享受高速、免费的 AI 编程体验。
