## 01\. 三大核心板块解析

为了搭建好这套环境，我们将第一章的内容拆解为三个紧密相关的板块。

第一个板块是电脑环境配置，主要涉及 Git 和 Node.js 等的安装。没有 Node.js，我们的网站程序就无法运行；没有 Git，我们的代码就无法保存和传输。

第二个板块是相关核心账号的注册，包括 GitHub、Vercel、Cloudflare 等 6 个核心服务。GitHub 用来存放你的代码资产，Vercel 用来向全世界展示你的网站，Cloudflare 用来保护和加速你的域名等等。

第三个板块是软件下载与配置，我们将安装 Cursor 编辑器和 Claude Code 智能助手，并配置国产的 GLM 4.7 模型来写代码。Cursor 是 AI 助手工作的办公室，Claude Code 是能够理解你指令的 AI 工程师，而 GLM 模型则是这位工程师的大脑。
