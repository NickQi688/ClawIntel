## 02\. 电脑环境配置

本章的主要任务是在电脑上安装开发所需的软件工具。这些工具是运行套壳视频站项目的地基。我们需要安装两类核心软件：Node.js（用于运行网页程序）和Git（用于保存代码历史）。配置好环境后，你的电脑将具备运行和修改项目代码的能力。

此处的教程分别针对 Mac 和 Windows 系统，内容较多，可从左侧大纲切换到自己的系统手册查阅。

在开始安装之前，请确认你的电脑满足以下配置要求。

Windows 用户需要使用 Windows 10 或更高版本的系统。

Mac 用户需要使用 macOS 10.15 或更高版本的系统。

为了保证软件流畅运行，电脑内存至少需要 8GB，推荐使用 16GB。

硬盘空间需要预留至少 10GB 的空闲空间。

此外，你需要连接稳定的互联网，以便下载所需的软件依赖。

如果网络环境有问题，可以私聊志愿者解决，不要在群里公开讨论。

如果你使用的是 Windows 电脑，请按照本节步骤操作。如果你是 Mac 用户，请直接跳到第 4 节。

4\. 常见问题与解决方法

在配置过程中可能会遇到一些意外情况，这里列出最常见的问题及解决方案。

4.1 命令提示符提示“不是内部或外部命令”

如果在 Windows 终端输入 git 或 node 命令后，系统提示“不是内部或外部命令”，这通常是因为环境变量没有生效。

解决方法是关闭当前的命令行窗口，重新打开一个新的窗口再次尝试。如果依然报错，可能需要卸载软件后重新安装，并确保在安装过程中勾选了“Add to PATH”选项（通常默认是勾选的）。

如果在 Mac 上执行命令时出现 Permission denied（权限被拒绝）的错误。 解决方法是在命令最前面加上 sudo 加空格，例如 sudo npm install。系统会要求输入开机密码以获取管理员权限。

4.3 下载速度极慢或超时

由于网络原因，直接从官方源下载可能会失败。

解决方法是确保使用了本教程提供的国内镜像源脚本（Gitee 链接）。如果依然卡顿，请检查网络连接是否稳定。
