## 03\. 相关账号注册

本章的主要任务是注册搭建视频站所需的各类核心服务账号，这些账号构成了项目的数字化基础设施。此处仅介绍核心服务的账号注册，后续在不同环节还需要注册的服务，会在对应章节说明。

此处不少账号需要具备良好的网络环境，如果网络不好或安装报错失败，可尝试换一个节点试试。

Google 账号在本项目中主要用于后续为网站添加“一键 Google 登录”功能。这是一种通用的第三方登录方式，能降低用户的注册门槛。

如果你已经拥有一个常用的 Google 账号（Gmail），可以直接使用该账号，无需重新注册。如果你还没有 Google 账号，或者希望使用一个新的账号来隔离开发环境，需要注册新账号。

GitHub 是全球最大的代码托管平台。你可以把它想象成一个云端的“代码仓库”，我们编写的所有项目代码都会存储在这里。它还负责记录代码的历史版本，防止文件丢失。同时，后续的 Vercel 部署服务也需要通过 GitHub 来获取代码。

打开浏览器，访问 GitHub 官方网站： https://github.com/

页面会显示一个欢迎动画，并提示你输入邮箱地址。在 Enter your email 处输入你的电子邮箱，点击 Continue（继续）。接着设置你的登录密码，要求密码具有一定的复杂度，输入后点击 Continue。然后设置一个唯一的用户名（Username），这个名字将作为你个人主页地址的一部分。

填写完信息后，点击 Create account（创建账号）。系统会向你的邮箱发送一封包含数字验证码的邮件。登录你的邮箱，找到验证码并填入 GitHub 页面，即可完成注册。

验证成功就回到登录页面，再次登录你的账号就可以了。

Vercel 是一个项目部署和托管平台。它的作用是将 GitHub 仓库中的代码自动构建并发布成一个可以在浏览器中访问的网站。它提供足够的免费额度供个人开发者使用。最终我们的产品代码会通过他转变成可在网络访问的链接地址。

此处选择 Hobby，用免费服务，暂时不用去付费买 Pro。

首次注册 Vercel 时，平台通常要求进行手机号验证以防止滥用。在弹出的界面中选择国家代码（中国大陆为 +86），输入你的手机号码，并填入接收到的短信验证码。验证通过后，账号即注册成功。

好了 注册成功，登录首页。

如果收不到短信验证码，请检查手机是否拦截了境外短信。如果提示该手机号已被使用，可以尝试通过邮箱方式找回账号，或者更换一个手机号进行验证。

Cloudflare 是一个网络基础设施平台。在本项目中，它承担三个重要角色：一是作为 DNS 解析服务商，帮助我们管理域名的指向；二是提供 R2 存储服务，用于廉价地存储生成的视频文件；三是提供邮箱路由功能，帮我们配置专业的域名邮箱。

Cloudflare 也在开发领域被戏称为“赛博菩萨”，因为他提供了足够大量的免费服务给大家。

访问 Cloudflare 中文官网或国际版官网： https://www.cloudflare.com/zh-cn/
