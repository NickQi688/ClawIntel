## 01\. 域名注册与解析

本章将完成域名的购买和配置工作。域名是网站的访问地址，就像你家的门牌号一样，用户通过域名才能找到你的网站。我们会使用 Spaceship 购买域名，然后将域名交给 Cloudflare 管理，最后配置一个专业的域名邮箱。

整个过程大约需要 30 分钟，花费在 10-20 美元之间。

Spaceship 是一家专业的域名注册商，我们推荐使用 Spaceship 的最主要原因是它支持支付宝付款，这对国内用户来说非常方便。相比其他域名注册商需要信用卡或 PayPal，Spaceship 降低了购买门槛。

其他推荐的域名注册商还包括 Namecheap, Dynadot 等等。

另外，我们不选择在国内服务商购买域名，是因为国内域名都需要备案，如果有国内需求也可以在国内购买并备案使用，本次航海不涉及这方面知识。

打开 Spaceship 官网后，在搜索框输入域名按回车后，就会出来搜索结果。

2\. 将域名解析切换到 Cloudflare

Cloudflare 是一家提供网站加速和安全防护的公司。我们把域名交给 Cloudflare 管理，主要有三个好处。

第一是 Cloudflare 提供免费的 CDN 加速，可以让全球用户访问你的网站时速度更快。

第二是 Cloudflare 提供基础的安全防护，可以抵御一些常见的网络攻击。

第三是 Cloudflare 的管理界面简单易用，后续添加子域名或修改解析记录都很方便。

DNS 是域名系统的缩写，它的作用是把域名翻译成服务器的 IP 地址。当用户在浏览器输入你的域名时，DNS 系统会告诉浏览器应该访问哪台服务器，我们现在要做的就是告诉 DNS 系统，你的域名由 Cloudflare 来管理。

首先登录 Cloudflare 账号（建议用 Github 或 Google 登录），进入后台后点击“加入域”

在弹出的输入框中输入你刚才购买的域名，注意不要输入 www 或 http 这些前缀，只需要输入纯域名即可。比如你购买的是 myaisite.com，就输入 myaisite.com。

AI训练阻止，选择不阻止，我们希望AI索引我们。

选择免费计划

点击继续后；如果解析记录为空，则可以点击最下面的“继续前往激活”。

但有时候会有几个解析记录已经存在，则需要全部删除后再继续。

点击继续后，Cloudflare 会显示两个服务器地址。这两个地址是 Cloudflare 分配给你的，把这两个地址记录下来，我们马上需要用到。

2.3 在 Spaceship 修改服务器地址

回到 Spaceship 网站，登录后进入域名管理页面。

https://www.spaceship.com/zh/application/domain-list-application/

找到你刚购买的域名，点击进入域名详情页。

在域名详情页找到 Nameserver 设置选项，点击修改。

选择自定义名称服务器，输入刚才从 Cloudflare 提供的两个服务器地址。

回到 Cloudflare，在刚才添加域名的页面点击完成，Cloudflare 会开始检查你的域名服务器是否已经修改成功。

这个检查过程可能需要几分钟到几个小时不等，我们不需要在此等待，可以先去做其他事情。

当域名成功接入 Cloudflare 后，你会收到一封确认邮件，邮件标题类似 您的域名现已在 Cloudflare 激活。这时再登录 Cloudflare，就能看到你的域名状态显示为已激活。

如果等待超过 24 小时域名还没有激活，首先检查你在 Spaceship 输入的域名服务器地址是否正确，即使多了一个空格或者少了一个字母，都会导致解析失败。
