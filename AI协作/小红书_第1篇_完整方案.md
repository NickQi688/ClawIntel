# 小红书第1篇：《当AI开始做小红书》

## 封面设计方案

### 视觉描述
**构图：** 左右对比分屏（50%:50%）

**左半部分（AI第一版）：**
- 背景：杂乱的渐变色（红+绿+蓝混在一起）
- 标题字体：Comic Sans MS（被设计师讨厌的字体）
- 排版：元素堆叠，没有对齐
- 配图：模糊的AI生成图（手指错乱）
- 整体感觉：业余、混乱、"AI审美"的刻板印象

**右半部分（AI第20版）：**
- 背景：专业的渐变色（紫色#6C5CE7到黄色#FFD93D）
- 标题字体：思源黑体 Bold，清晰醒目
- 排版：网格对齐，留白得当
- 配图：精美的扁平化插画
- 整体感觉：专业、精美、小红书爆款风格

**中间大标题：**
```
当AI开始做小红书
做了3小时封面
```

**底部小字：**
```
第一版 vs 第20版
```

### 配色方案
```css
主色1：#FFD93D（亮黄）
主色2：#6C5CE7（深紫）
辅助：#FF6B9D（亮粉）
文字：#FFFFFF（白色）或 #2D3436（深灰）
```

### 尺寸规格
- 小红书封面：1242×1660 像素
- 比例：3:4
- 分辨率：72dpi（网页）或 300dpi（印刷）

## 生成方案（三选一）

### 方案1：Gemini API（推荐，自动）
**需要：**
- Gemini API Key
- 运行生成脚本

**步骤：**
```bash
export GEMINI_API_KEY='你的密钥'
python3 generate_xhs_cover_v2.py
```

**优点：** 一键生成，质量高
**缺点：** 需要API Key

---

### 方案2：Canva/稿定设计（手动，快速）
**步骤：**
1. 打开 canva.com 或 gaoding.com
2. 搜索"小红书封面"模板
3. 选择3:4比例模板
4. 修改文字和配色（黄紫撞色）
5. 左边放丑图，右边放美图
6. 下载保存

**优点：** 免费，快速，有模板
**缺点：** 需要手动操作

---

### 方案3：找设计师（外包）
**需求文档：**
```
需求：小红书封面设计
主题：AI做封面3小时的进化史
构图：左右对比（第一版 vs 第20版）
风格：扁平化插画，科技感，有趣
配色：亮黄(#FFD93D) + 深紫(#6C5CE7)
尺寸：1242×1660像素
预算：50-100元
```

**优点：** 质量最好
**缺点：** 需要时间和费用
