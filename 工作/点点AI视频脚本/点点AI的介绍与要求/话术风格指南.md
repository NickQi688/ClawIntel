# 点点AI视频脚本 - 话术风格指南

> 更新时间：2026-02-09
> 来源：甲方Brief + 参考脚本分析
> 用途：指导脚本创作，确保有人情味、自然植入新功能

---

## ⭐ 甲方核心要求（必须遵守）

### 🎨 语言风格（来自甲方Brief）

| ✅ 要求 | ❌ 避免 |
|--------|--------|
| **轻快活泼** | 正经严肃 |
| **有温度、有梗** | 长篇大论爹味发言 |
| **分段式发言**（有人感） | 机械式陈述 |
| **活人感** | 冰冷的AI腔 |

### 🎭 情感基调（来自甲方Brief）

```
焦虑 → 安心
困惑 → 清晰
繁琐 → 简单
```

**叙事视角**：
- 第二人称"你"为主（让用户有代入感）
- 第一人称"我"（真实体验分享）

---

## 🎯 人情味话术核心原则

### 1. 情绪化开头，抓住注意力
**❌ 机械式开头**：
```
想带老人出去玩，但不知道去哪好
```

**✅ 人情味开头**：
```
想带爸妈出去玩
但他们总说"不去了，走不动"
其实不是不想玩
是怕太累、怕走太多路
```

**技巧**：
- 用具体对话代替抽象描述
- 展示内心活动，而非陈述事实
- 用"其实"揭示真实原因

---

### 2. 自嘲式幽默，拉近距离
**❌ 过于正式**：
```
带老人出游需要注意体力问题
```

**✅ 自嘲幽默**：
```
带爸妈出去
结果他们走两步就累了
我也太不会选地方了
```

**技巧**：
- 暴露自己的"笨"或"不会"
- 让观众产生"我也这样"的共鸣
- 朋友吐槽式的语气

---

### 3. 情感词+程度副词，增强温度
**常用情感词**：
- 真的、真的真的、最最最喜欢
- 居然、竟然、居然还可以这样
- 超靠谱、绝了、救命

**示例**：
```
❌ 点点总结了推荐
✅ 点点立刻总结了推荐，超靠谱！
❌ 我很喜欢这个功能
✅ 我个人最最最喜欢的是它竟然还有导航栏
```

---

### 4. 口语化表达，避免书面语
**❌ 书面语**：
```
该功能可以帮助用户节省时间
```

**✅ 口语化**：
```
再也不用自己找资料扒半天
一步到位不踩坑
```

**技巧**：
- 用"再也不用"、"直接抄作业"
- 用"扒半天"、"踩坑"等网络用语
- 短句为主，节奏明快

---

### 5. 拟人化+朋友语气
**❌ 工具化描述**：
```
点点AI是一个智能助手
```

**✅ 拟人化+朋友语气**：
```
点点做你的"专属军师"
小红书自己的AI
真人经验总结
```

**技巧**：
- 用"军师"、"搭子"、"朋友"等拟人化词汇
- 强调"小红书自己的"、"真人经验"
- 让点点有"温度"，不是冰冷的AI

## ✨ 新功能植入指南

```

输入你的需求

```

"上海周边，适合3岁宝宝，不累、有教育意义"

- 地图标点能力（可执行性强）
- 实时信息（时效性强）

## 🚀 创作检查清单

写脚本时，检查是否有人情味：

- [ ] **开头情绪化**：不是陈述句，有具体场景/对话
- [ ] **有自嘲/吐槽**：暴露自己的"不会"或"笨"
- [ ] **情感词丰富**：真的、超、绝了、最最最
- [ ] **口语化表达**：不用"该功能可以帮助用户"
- [ ] **朋友语气**：像给朋友推荐，不是推销
- [ ] **植入新功能**：攻略模式、拍照识图、总结笔记等
- [ ] **强调真实感**：真人经验、真实体验、超靠谱
