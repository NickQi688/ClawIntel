# 固定工作流 - 200条/月漫画风视频
# 蜀座-停车场烧烤项目标准操作流程

##  工作流概览

**目标：** 1个月交付200条漫画风AI视频
**策略：** 即梦Agent模式 + 可灵视频 + 剪映合成
**速度：** 25-30分钟/条
**成本：** 98-196元/月

---

##  完整工作流（4步）

### 第0步：准备工作（一次性，2-3小时）

#### 任务清单
- [ ] 收集客户提供的照片素材（15-20张）
- [ ] 生成男主角三视图（即梦图片生成）
- [ ] 生成女主角三视图（可选）
- [ ] 测试即梦Agent模式（生成6个场景）
- [ ] 测试可灵视频生成
- [ ] 制作第一条完整视频

#### 详细步骤

**1. 收集照片素材**
```
必需照片：
- 门头正面 2张
- 店铺街景 2张
- 招牌菜品（五花肉、牛肋条）各3张（生/烤中/熟）
- 店内环境 3张
- 炭火烤架 2张

拍摄要求：
- 时间：晚上20:00-23:00
- 光线：暖色灯光
- 清晰：不模糊、不抖动
- 招牌：能看到"蜀座停车场烧烤"
```

**2. 生成角色三视图**
```
工具：即梦AI - 图片生成
网址：https://jimeng.jianying.com/

操作：
1. 打开即梦
2. 选择"图片生成"
3. 模型：图片4.0，比例9:16
4. 输入提示词（见下方）
5. 点击生成
6. 选择最满意的图片
7. 导出保存

男主角提示词：
一键生成角色三视图，全身，
（25岁男性，戴黑框眼镜，穿深蓝色连帽卫衣，
黑色休闲裤，背黑色双肩包，短发，略疲惫的表情）。
三视图人物IP设计：正视图，侧视图，后视图，
8K高清，治愈系插画风格，柔和笔触，
温馨色调，日系漫画风格。

保存位置：
素材库/角色设计/男主角_三视图.png
```

**3. 测试Agent模式**
```
工具：即梦AI - Agent模式

操作：
1. 打开即梦 → 选择"Agent模式"
2. 上传"男主角_三视图.png"
3. 输入场景提示词（从工作流文档复制）
4. 立即生成
5. 查看效果
6. 继续对话生成第2个场景
7. 确认角色一致性

测试场景（6个）：
- 场景1：下班路上
- 场景2：路过烧烤店
- 场景3：走进店内
- 场景4：烤肉上桌
- 场景5：咬下第一口
- 场景6：店铺夜景

时间：10-20分钟
```

**4. 测试视频生成**
```
工具：可灵AI
网址：https://app.klingai.com/cn/

操作：
1. 选择场景1的图片
2. 选择"生成视频"
3. 输入动作提示词："烟雾缓慢上升，镜头极慢推进"
4. 生成长度：5秒
5. 生成2个版本
6. 下载最自然的版本

时间：5分钟
```

**5. 制作第一条完整视频**
```
工具：剪映手机版

操作：
1. 导入6个场景视频
2. 按顺序排列
3. 添加转场（叠化0.5秒）
4. 调整时长（每片2-3秒，总12-15秒）
5. 添加音效（滋滋声）
6. 添加BGM（治愈系吉他曲，音量15-20%）
7. 导出1080P

时间：15分钟
```

**准备工作总时间：** 2-3小时

```
即梦首页 → 选择"Agent模式"

**2. 上传角色三视图**
```

第1次对话：

第2次对话（继续）：

第3次对话（继续）：

...继续直到生成完所有场景

优势：
- 不需要重新上传角色
- 保持人物一致性
- 速度快（10-20秒/场景）

### 2. 场景图片生成视频（10分钟/组）

#### 工具：可灵AI

#### 操作流程

**1. 批量上传图片**
```
操作：
1. 打开可灵AI
2. 选择"图生视频"
3. 一次性上传6个场景图片
```

**2. 统一添加动作提示词**
```
动作提示词库（根据场景选择）：

场景类型1（深夜治愈）：
"烟雾缓慢上升，灯光轻微闪烁，镜头极慢推进"

场景类型3（烤肉过程）：
"油脂缓慢流动，烟雾轻柔上升，滋滋冒油动态"

场景类型4（炭火人间）：
"炭火微微跳动，烟雾轻柔飘动，火星缓慢飞溅"

场景类型5（四季烧烤）：
"烟雾轻柔上升，光影缓慢变化"

场景类型6（思念味道）：
"烟雾缓慢飘动，灯光温暖闪烁，整体偏安静"
```

**3. 批量生成视频**
```
操作：
1. 选择图片1 → 输入动作提示词 → 生成
2. 选择图片2 → 输入动作提示词 → 生成
3. ...依次生成所有图片

参数设置：
- 长度：5秒
- 版本：生成2个，选择最好的
- 下载：保存到对应文件夹
```

**生成6个场景视频时间：** 8-10分钟

### 3. 剪映快速合成（15分钟/条）

#### 工具：剪映手机版

#### 标准流程（10步）

**1. 导入视频片段**
```
打开剪映 → 开始创作 → 导入6个场景视频
```

**2. 按顺序排列**
```
拖动视频片段按故事顺序排列：
场景1-1 → 场景1-2 → 场景1-3 → 场景1-4 → 场景1-5 → 场景1-6
```

**3. 添加转场**
```
点击片段之间 → 选择"叠化" → 设置时长0.5秒
所有片段之间都添加
```

**4. 调整时长**
```
每个片段：
- 选择片段 → 调整时长为2-3秒
- 删除多余帧
- 总时长：12-15秒
```

**5. 添加音效**
```
音频 → 音效 → 搜索"滋滋声"
添加到烤肉场景
音量：30-50%

其他音效：
- 碰杯声（举杯场景）
- 环境音（街景场景）
- 炭火声（炭火场景）
```

**6. 添加BGM**
```
音频 → 音乐 → 搜索"治愈系"
选择BGM → 添加到时间轴
音量：15-20%（不要盖过音效）

BGM分类：
- 治愈系：钢琴曲、吉他民谣
- 节奏感：轻快电子、爵士
- 生活化：环境音、自然音
```

**7. 预览调整**
```
播放完整视频 → 检查：
- 时长是否合适
- 转场是否流畅
- 音量是否平衡
- 整体感觉是否治愈
```

**8. 导出设置**
```
导出 → 分辨率：1080P → 格式：MP4 → 帧率：30fps
```

**9. 保存到相册**
```
导出完成后自动保存到相册
```

**10. 命名保存**
```
文件命名格式：
场景类型1-深夜治愈-001.mp4
场景类型1-深夜治愈-002.mp3
...

便于后续管理和发布
```

**剪辑时间：** 15分钟/条（首条）
**有模板后：** 10分钟/条

---

### 4. 发布（5分钟/条）

#### 抖音发布流程

**1. 上传视频**
```
打开抖音 → 点击"+" → 上传视频 → 选择刚导出的视频
```

**2. 输入文案**
```
标题文案库（复制粘贴）：

场景类型1：
标题：成年人的治愈时刻
文案：下班以后，这条街才真正热闹起来

场景类型2：
标题：一个人吃烧烤，不叫孤独
文案：一个人吃烧烤，不叫孤独，叫享受

场景类型3：
标题：看着肉变熟，太治愈了
文案：古法新鲜烤，不烤冷冻肉

场景类型4：
标题：人间烟火气，最抚凡人心
文案：炭火燃起，才是夜的开始

场景类型5：
标题：春天适合想念（根据季节调整）
文案：春天适合想念 / 夏天适合遇见 / 秋天适合收获 / 冬天适合团聚

场景类型6：
标题：有些味道，和某个人有关
文案：原来想念是有味道的
```

**3. 添加话题标签**
```
必带：
#烧烤 #西安烧烤 #深夜食堂

可选：
#治愈 #独食 #烟火气 #古法烤肉 #夜宵

根据场景类型选择2-3个
```

**4. 添加定位**
```
位置 → 搜索"西安" → 选择店铺所在区域 → 确定
```

**5. 定时发布**
```
发布设置 → 定时发布 → 选择时间：
最佳时段：20:00-23:00
推荐：20:30、21:00、21:30
```

**6. 确认发布**
```
点击"发布" → 等待审核 → 发布成功
```

**发布时间：** 5分钟/条

##  批量制作优化（第2周开始）

### 模板化流程

**建立剪映模板：**
```
第1条：完整制作（15分钟）
第2条：复制项目 → 只修改文案和BGM（10分钟）
第3条及以后：复制项目 → 只修改文案和BGM（10分钟）

操作：
1. 打开剪映 → 找到第1条项目
2. 点击"复制" → 创建副本
3. 删除原来的视频片段
4. 导入新的视频片段
5. 保持转场、音效不变
6. 只修改BGM（可选）
7. 导出
```

**批量产出技巧：**
```
同一场景类型，做多条：

1. 一次性生成6个场景（3分钟）
2. 一次性生成6个视频（8分钟）
3. 剪映合成第1条（15分钟）
4. 剪映复制项目做第2条（10分钟）
5. 剪映复制项目做第3条（10分钟）
...

优势：
- 场景只需要生成一次
- 只需换文案和BGM
- 大幅提升效率
```
