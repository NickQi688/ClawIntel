##  图片生成工具

### 1. 即梦 AI（主力推荐）

**网址：** https://jimeng.jianying.com/

**优势：**
-  每天随机送60-100积分（免费）
-  图片和视频都能做
-  Agent模式支持角色一致性
-  付费可去水印（性价比高）
-  与剪映同生态，无缝衔接

**适用场景：**
- 漫画风图片生成
- 角色三视图生成
- Agent批量生成场景

**核心功能：**

#### 图片生成
- 模型：图片4.0
- 比例：9:16（竖屏）或16:9
- 分辨率：标清2K（会员4K）
- 生成时间：10-20秒/张

#### Agent模式（关键！）
```
操作流程：
1. 打开即梦 → 选择"Agent模式"
2. 上传角色三视图
3. 输入场景提示词
4. 立即生成
5. 继续对话生成下一场景（无需重新上传角色）

优势：
- 保持角色一致性
- 批量生成6-10个场景
- 速度比单独生成快3-5倍
```

**使用技巧：**
- 上午使用积分更充足
- 生成3-5个版本，选择最好的
- 保存满意的图片为"画风参考"
- 后续生成参考此画风

**成本：**
- 免费：每天60-100积分（约3-5张图）
- 会员：98元/月，无限生成
- 推荐充值：先免费试用，效果好再付费

##  视频生成工具

### 1. 可灵 AI（主力推荐）

**网址：** https://app.klingai.com/cn/

**优势：**
-  免费（有积分限制）
-  质量很高，动态自然
-  图生视频效果好
-  支持多种动作提示
-  付费会员可去水印

**适用场景：**
- 真实照片+AI动态
- 漫画风视频生成
- 所有图生视频需求

**核心功能：**

#### 图生视频
```
操作流程：
1. 上传图片（或用即梦生成的图）
2. 输入动作提示词（如"烟雾缓慢上升"）
3. 生成长度：5秒
4. 生成2-3个版本
5. 选择最自然的版本
```

#### 动作提示词库
```
通用类：
- 烟雾缓慢上升
- 灯光轻微闪烁
- 镜头极慢推进
- 镜头缓慢拉远

烤肉类：
- 油脂缓慢流动
- 肉表面微微冒热气
- 滋滋冒油动态

氛围类：
- 炭火微微跳动
- 火星缓慢飞溅
- 顾客剪影轻微移动
```

**使用技巧：**
- 动作提示词越简单越自然
- 避免复杂动作（容易崩坏）
- 生成多个版本选最好的
- 5秒长度最稳定

**成本：**
- 免费：每天有积分限制
- 会员：98元/月（推荐）

## ✂️ 剪辑工具

### 剪映（手机版）

**下载：** App Store / 应用商店搜索"剪映"

**优势：**
-  完全免费
-  功能齐全
-  操作简单
-  音效库丰富
-  BGM库丰富

**核心功能：**

#### 基础剪辑
- 视频拼接
- 转场效果（叠化0.5秒）
- 调整速度
- 删除多余帧

#### 音频处理
- 添加BGM（音量15-20%）
- 添加音效（滋滋声、碰杯声等）
- 音量调整
- 淡入淡出

#### 导出设置
- 分辨率：1080P
- 格式：MP4
- 帧率：30fps

**使用技巧：**
1. 先导入所有视频片段
2. 按顺序排列
3. 添加转场（所有片段之间）
4. 调整每个片段时长（2-3秒）
5. 添加音效和BGM
6. 最后导出

**制作时间：**
- 首条：15分钟
- 有模板后：10分钟

##  效率优化技巧

### 1. 积分管理

**即梦积分优化：**
- 上午使用积分更充足
- 一次性生成6-10个场景（Agent模式）
- 避免频繁生成单张图
- 保存满意的提示词

**可灵积分优化：**
- 集中时间段批量生成
- 一次上传多个图片
- 选择5秒长度（最稳定）
- 生成2-3个版本即可

---

### 2. 批量制作技巧

**图片生成：**
```
优化前：
单独生成每张图 → 每张都输入完整提示词 → 速度慢

优化后：
使用Agent模式 → 上传一次角色 → 批量生成6-10个场景 → 速度提升3-5倍
```

**视频生成：**
```
优化前：
单个生成 → 每个都要上传 → 重复操作多

优化后：
批量上传 → 统一添加动作提示词 → 一次性生成6-10个
```

**剪辑流程：**
```
优化前：
每条视频从头开始 → 重复操作多

优化后：
建立模板 → 复制项目 → 只修改文案和BGM → 速度提升50%
```

---

### 3. 模板化管理

**建立提示词模板库：**
```
文件结构：
提示词模板/
├── 场景类型1-深夜治愈/
│   ├── 场景1-1.txt
│   ├── 场景1-2.txt
│   └── ...
├── 场景类型2-独食浪漫/
│   └── ...
└── 场景类型3-烤肉过程/
    └── ...

使用方法：
直接复制提示词 → 粘贴到即梦 → 快速生成
```

**建立文案库：**
```
文案库.txt

标题分类：
- 深夜治愈类：10个
- 独食浪漫类：10个
- 烤肉过程类：10个
- 炭火人间类：10个
- 四季烧烤类：10个
- 思念味道类：10个

使用方法：
直接复制标题 → 粘贴到抖音 → 快速发布
```

**建立音效库：**
```
音效库/
├── 滋滋声.mp3
├── 碰杯声.mp3
├── 环境音.mp3
└── 炭火声.mp3

BGM库/
├── 治愈系/
│   ├── 钢琴曲1.mp3
│   └── 吉他民谣1.mp3
├── 节奏感/
│   ├── 轻快电子1.mp3
│   └── 爵士1.mp3
└── 生活化/
    ├── 环境音1.mp3
    └── 自然音1.mp3
```
