# AI视频项目 - 沟通规范和原则

## 项目信息
**客户**：蜀座停车场烧烤
**交付周期**：200条/月（漫画风治愈系视频）
**制作方式**：图生图（基于店铺实拍图）+ AI视频生成

---

## 提示词编写原则

### 图片提示词
**错误示范**：
```
女主人公走向烧烤店门头，"蜀座停车场烧烤"招牌可见，窗户透出暖黄灯光...
```
❌ 描述了太多具体元素（门头、窗户、招牌），但参考图里可能没有这些

**正确示范**：
```
女主人公穿厚外套，走向店门，冷蓝色调，雾气氛围，治愈系插画
```
✅ 只写：人物动作、氛围、风格
✅ 不写：店铺招牌、窗户、这些参考图里有的元素

**核心原则**：
- 简洁直接
- 只描述人物动作、氛围、风格
- 不描述具体场景元素（会用图生图保留参考图里的元素）
- 不要瞎编参考图里没有的东西

### 视频提示词
**格式**：
```
人物走向门口→推门进入→坐下吃烧烤，色温从冷蓝变暖黄
```

**要点**：
- 用箭头表示镜头流程
- 简单描述动作和变化
- 不啰嗦

## 视频制作规范

### 统一参数
- 时长：8-12秒
- 比例：9:16
- 风格：治愈系插画
- 字幕：打字机效果，金色/暖黄色，居中偏下

### 图数量
- 1-2张/视频（不需要太多）
- 简单场景1张足够
- 有空间变化时用2张（室外→室内）

### 季节元素（冬天）
**必须体现**：
- 人物穿着：厚外套、围巾、帽子（室外）
- 室内：毛衣、卫衣（有暖气）
- 蒸汽：食物冒热气、呼吸的白气
- 氛围：冷暖对比（室外冷蓝→室内暖黄）

**差异化策略**：
- 对标都是夏天，你是冬天，形成独特记忆点
- 冬天的治愈感更强（冷夜中的温暖烧烤）

### 店铺元素保留
**图生图模式**：
- 基于店铺实拍图（门头、室内、菜品）
- 必须保留：门头招牌、室内装修风格、座位布局
- 提示词不写这些元素（参考图里已经有了）

## 文案创作原则

### 核心方向
**情绪共鸣**（好的坏的都可以）：
- 好的：温暖、治愈、幸福、相聚
- 不好的：疲惫、怀念、不容易、孤独
- 关键：能引起共鸣

### 文案风格
- 接地气、口语化
- 可加入地域元素（西安等）
- 简短有力（8-12秒视频，文案不超过20字）

### 结构模式
- 开头：场景/状态描述
- 中间：情感升华
- 结尾：提问互动 OR 情感金句

### 顾客画像多样化
- 一家三口、一家四口
- 三代同堂（爷爷奶奶）
- 朋友聚餐、闺蜜聚餐
- 多年不见的老友
- 公司聚餐
- 情侣
- 独食

## 工作流程

### 单个视频制作流程
1. 确定文案
2. 准备参考图（店铺实拍）
3. 编写图片提示词（简洁版）
4. 编写视频提示词（简洁版）
5. 图生图生成图片
6. 视频生成（首尾帧或单图）

### 提示词确认
- 先给用户看提示词
- 确认无误后再生成
- 有问题及时调整

## 关键经验总结

### 踩过的坑
1. ❌ 提示词描述了窗户，但参考图没有窗户 → AI硬加一个很怪
2. ❌ 提示词太啰嗦，描述了太多元素 → 用户图生图会保留，不需要我写
3. ❌ 没预设道具（如啤酒） → 后面要做碰杯动作时发现前面没埋伏笔

### 正确做法
1. ✅ 参考图里有什么，提示词就不写什么
2. ✅ 提示词只写：人物动作、氛围、风格
3. ✅ 如果后面要做动作（如碰杯），前面提示词就要预设道具（啤酒）

### 人物连贯性
- 同一个视频里的主人公要保持一致
- 如果是"从室外到室内"的场景，起始帧和结束帧要是同一个主人公
- 服装变化要合理（厚外套→毛衣）
