# 漫画风AI视频完整工作流 - 优化版
# 基于即梦Agent模式的高效制作方案

##  核心发现

核心发现：
即梦有一个**Agent模式**，可以：
1. 上传角色三视图 → 保持人物一致性
2. 输入分镜提示词 → 批量生成不同场景
3. 自动生成视频 → 大幅提升效率

**这对我们的意义：**
-  解决了"人物不一致"的痛点
-  批量生成场景速度提升3-5倍
-  200条/月完全可行

---

##  整体策略调整

### 新工作流核心思路

**1. 创建固定角色（一次性，2小时）**
- 设计1-2个治愈系主角（年轻上班族情侣）
- 生成三视图（正、侧、后）
- 保存为"标准角色"

**2. 基于角色批量生成场景（每天）**
- 用Agent模式上传角色
- 输入不同场景提示词
- 自动生成6-10个场景图片
- 一键生成视频

**3. 剪映快速合成（15分钟/条）**
- 导入视频片段
- 添加BGM和音效
- 导出发布

##  优化后的标准工作流

### 第0步：准备工作（一次性，2-3小时）

#### 1. 生成角色三视图
**工具：** 即梦AI图片生成

**操作流程：**
1. 打开即梦：https://jimeng.jianying.com/
2. 选择"图片生成"
3. 模型：图片4.0，比例9:16（竖屏）
4. 输入角色三视图提示词
5. 点击生成
6. 选择最满意的图片
7. 导出保存

**保存位置：**
```
素材库/
└── 角色设计/
    ├── 男主角_三视图.png
    └── 女主角_三视图.png（可选）
```

---

#### 2. 收集店铺照片（客户提供的）
```
素材库/
└── 店铺照片/
    ├── 门头正面.jpg
    ├── 店铺街景.jpg
    ├── 五花肉烤肉.jpg
    └── ...
```

**基于店铺照片生成参考图：**
- 用即梦"图生图"功能
- 将真实照片转为漫画风
- 保存为"标准画风参考"
- 后续所有场景基于此画风

### 1. Agent模式批量生成场景（每天）

#### 工具：即梦Agent模式

**操作流程：**

**1. 打开Agent模式**
- 即梦首页 → 选择"Agent模式"

**2. 上传角色三视图**
- 点击上传
- 选择之前保存的"男主角_三视图.png"

**3. 输入第一个场景提示词**
```
25岁男性戴黑框眼镜穿深蓝色连帽卫衣，
背着黑色双肩包，夜晚走在西安街头背影，
道路左侧是西安古城墙背景，右侧是现代化高楼大厦灯光，
路灯暖黄光，远处大雁塔剪影，深蓝夜空，
孤独但有希望的氛围，治愈系插画风格，
柔和笔触，9:16竖屏，镜头缓慢推进
```

**4. 立即生成**
- 等待10-20秒
- 查看生成效果

**5. 继续对话生成下一场景**
- 不需要重新上传角色
- 直接输入下一个场景提示词
- 继续生成

**批量生成技巧：**
- 一次对话可以生成6-10个场景
- 保持角色一致性
- 速度比单独生成快3-5倍

**生成1个场景的时间：** 10-20秒
**生成6个场景的时间：** 2-3分钟

---

### 2. 场景图片生成视频（10分钟/组）

#### 工具：即梦视频生成 / 可灵AI

**操作流程：**

**1. 选择生成的场景图片**
- 点击进入某个场景
- 选择"生成视频"

**2. 输入动作提示词**
```
烟雾缓慢上升，灯光轻微闪烁，镜头极慢推进
```

**动作提示词库：**
```
通用类：
- 烟雾缓慢上升
- 灯光轻微闪烁
- 镜头极慢推进
- 镜头缓慢拉远

烤肉类：
- 油脂缓慢流动
- 肉表面微微冒热气
- 滋滋冒油动态

氛围类：
- 炭火微微跳动
- 火星缓慢飞溅
- 顾客剪影轻微移动
```

**3. 生成视频**
- 长度：5秒
- 生成2-3个版本
- 选择最自然的版本

**4. 下载视频**
- 保存到对应文件夹

**生成6个场景视频的时间：** 8-10分钟

---

### 3. 剪映快速合成（15分钟/条）

#### 工具：剪映（手机版）

**操作流程：**

**1. 导入视频片段**
- 打开剪映
- 导入6个场景视频
- 按顺序排列

**2. 拼接视频**
- 每个片段2-3秒
- 总时长12-15秒
- 删除多余帧

**3. 添加转场**
- 选择"叠化"效果
- 时长0.5秒
- 所有片段之间添加

**4. 添加音效**
```
必备音效：
- 滋滋声（烤肉场景）
- 碰杯声（举杯场景）
- 环境音（街景场景）
- 炭火声（炭火场景）

音量设置：
- BGM：15-20%
- 音效：30-50%
```

**5. 添加BGM**
```
BGM分类：
- 治愈系：钢琴曲、吉他民谣
- 节奏感：轻快电子、爵士
- 生活化：环境音、自然音
```

**6. 导出视频**
