# 蜀座-蜀座停车场烧烤 纯情绪价值视频方案（无营销版）


##  核心定位

### 内容原则
- **100% 纯情绪价值** - 不含任何营销信息
- **先做内容，再做品牌** - 让观众记住情感体验
- **去商业化** - 打造真诚、治愈的内容氛围

### 不包含的内容
-  店铺地址
-  联系电话
-  优惠信息
-  "欢迎光临"
-  "我们在XX路"
-  价格信息
-  领券/活动
-  导流话术

### 包含的内容
-  情绪共鸣（治愈、孤独、怀念）
-  视觉美学（美食特写、氛围场景）
-  城市特色（西安古今结合）
-  品牌元素（自然融入的场景）

### 为什么无营销反而有效？
1. **降低防备心** - 观众不会反感
2. **增加完播率** - 纯内容更吸引人
3. **引发搜索** - 好奇会主动搜索店铺
4. **建立信任** - 真诚内容更容易被记住
5. **自然传播** - 情绪内容更易转发

##  3 种情绪类型

### 1. 治愈系
**核心情绪：** 温暖、安慰、疗愈
**适合场景：** 深夜、疲惫、独处
**目标观众：** 上班族、独居青年、夜宵党

### 2. 孤独系
**核心情绪：** 孤独但不寂寞、自我享受
**适合场景：** 一个人吃饭、深夜食堂
**目标观众：** 独居者、内向者、喜欢独处的人

### 3. 怀念系
**核心情绪：** 怀念、回忆、时光流逝
**适合场景：** 老友、故乡、旧时光
**目标观众：** 异乡人、怀旧党、情感丰富的人

##  制作工具和流程

### 推荐工具组合

**图片生成：**
- 即梦（主要，免费）
- 豆包（备选）

**视频生成：**
- 可灵 AI（主要，免费）
- 即梦视频（备选）

**剪辑工具：**
- 剪映（手机+电脑）

### 统一制作流程

#### 第 1 步：图片生成（15-20 分钟）
```
1. 复制对应方向的提示词
2. 即梦生成 3-5 张场景图
3. 选择最好的版本
4. 保持人物一致（固定角色功能）
```

#### 第 2 步：视频生成（10-15 分钟）
```
1. 可灵 AI 上传图片
2. 输入动作提示词（如"烟雾缓缓上升"）
3. 生成 5 秒视频片段
4. 下载所有片段
```

#### 第 3 步：剪辑合成（10-15 分钟）
```
1. 剪映导入 3-5 个片段
2. 拼接（总时长 10-15 秒）
3. 添加转场（叠化 0.5 秒）
4. 加音效（滋滋声、环境音）
5. 加 BGM（15-20% 音量）
6. 导出 1080P
```

#### 第 4 步：发布（5 分钟）
```
1. 抖音上传视频
2. 输入标题（简短有力）
3. 添加话题标签
4. 添加西安定位
5. 发布时间：20:00-23:00
```

**总耗时：**
- 首次：50-60 分钟/条
- 后续：35-45 分钟/条

##  无营销的优势

### 1. 降低观众防备心

**有营销的视频：**
- 观众："又是广告" → 划走
- 完播率低，互动少

**无营销的视频：**
- 观众："这也太治愈了" → 看完
- 完播率高，互动多

### 2. 增加完播率

**纯情绪内容：**
- 没有商业打断
- 观众愿意看完
- 算法推荐更多

### 3. 引发主动搜索

**心理学原理：**
- 被情绪打动 → 产生好奇 → 主动搜索
- 比"硬广"记忆更深

**现实案例：**
- 很多爆款视频无营销
- 观众在评论区求店名
- 形成自发传播

### 4. 建立真诚形象

**品牌印象：**
- "这家店很真诚"
- "不为赚钱，只为分享"
- 反而更受欢迎

### 5. 自然传播

**分享动力：**
- 情绪内容易共鸣
- 愿意转发给朋友
- "这个视频太治愈了，你看"
