# 场景素材库使用指南

## 目录结构

```
assets_library/
├── 门头/           # 店铺门头、招牌场景
│   ├── 门头正1.jpg
│   ├── 门头正2.jpg
│   └── 门头侧1.jpg
├── 室内/           # 室内场景、座位区
│   ├── 室内全景1.jpg
│   └── 座位区1.jpg
├── 人物/           # 人物参考图
│   └── 时尚美女1.jpg
├── 食物/           # 食物特写
│   └── 烤串1.jpg
└── 其他/           # 其他场景
    └── 街景1.jpg
```

## 命名规则

**格式**：`{场景名}{角度}{编号}.jpg`

**示例**：
- `门头正1.jpg` - 门头正面第1张
- `门头侧2.jpg` - 门头侧面第2张
- `室内全景1.jpg` - 室内全景第1张

## 添加素材

1. 将图片放到对应目录
2. 按命名规则重命名
3. 更新 `video_assets_library.json` 索引

## 自动匹配逻辑

video-workflow skill会根据分镜头描述自动匹配素材：

| 描述关键词 | 调用目录 |
|-----------|---------|
| 门头、门口、店面 | 门头/ |
| 室内、店内、座位 | 室内/ |
| 人物、美女、老板 | 人物/ |
| 烤串、烧烤、食物 | 食物/ |
| 街景、夜景、背景 | 其他/ |

## 使用示例

### 自动匹配
```
你："门头处，时尚美女吃烤串"
skill → 自动检测"门头" → 调取 门头/门头正1.jpg
```

### 手动指定
```
你："用门头侧1"
skill → 直接调取 门头/门头侧1.jpg
```

### 无需素材
```
你："纯人物特写"
skill → 不调用素材库，纯提示词生成
```

## 注意事项

- 图片质量要高（至少1080p）
- 光线要充足
- 角度要多样（正、侧、俯等）
- 定期清理不用的素材
