## Obsidian 是什么？

一句话：**本地知识库，你的第二大脑。**

### 核心优势

**第一，数据完全属于你。**

Obsidian 的笔记就是一堆 Markdown 文件，存在你电脑里。即使 Obsidian 明天倒闭，你的笔记还在。换个编辑器，继续用。

**第二，双向链接改变思维方式。**

传统笔记是线性的（像一本书），Obsidian 的笔记是网状的（像大脑）。

一个概念可以同时出现在十篇笔记里，它们自动互相关联。你不再是在"记录"，而是在"构建"一个知识网络。

**第三，和 AI 是绝配。**

Obsidian 的笔记是本地 Markdown 文件。Claude Code、OpenCode 这些 AI 工具，天生就能直接读取你的笔记。

## 第二部分：创建你的第一个知识库

### 什么是 Vault（知识库）？

**说人话：一个文件夹，装你的所有笔记。**

### 创建步骤

1. 打开 Obsidian
2. 点击"创建新库"
3. 选择一个文件夹位置（建议放在文档里，比如"我的知识库"）
4. 给它起个名字（比如"数字分身系统"）
5. 点击创建

**就这么简单。**

### 核心概念

**文件夹结构：**
```
我的知识库/
├── 笔记1.md
├── 笔记2.md
├── 图片/
└── 附件/
```

**Markdown 格式：**
- Obsidian 使用 Markdown 格式
- 纯文本，到处都能打开
- 不被任何工具锁定

## 第三部分：核心功能快速上手

### 1. 创建笔记

**方式1：快捷键**
- Mac：`Cmd + N`
- Windows：`Ctrl + N`

**方式2：点击图标**
- 点击左侧文件夹图标
- 点击"新建文件"

### 2. 双向链接（核心功能）

**怎么用？**

在笔记中用 `[[双括号]]` 框住任何词：

```markdown
今天学习了[[数字分身]]，很有意思。
```

效果：
- 自动创建一个叫"数字分身"的笔记
- 两个笔记自动关联
- 点击链接就能跳转

**威力在哪里？**

**传统笔记：**
- 你写了一篇关于"用户增长"的笔记，放在"运营"文件夹
- 很少再去看它

**Obsidian 思维：**
- 你写了一篇关于"用户增长"的笔记
- 它和[[AARRR模型]]有关
- 它和[[北极星指标]]有关
- 它和[[增长黑客]]这本书有关

**碎片信息，自动汇总。**

### 3. 反向链接面板

在笔记右侧，有一个"反向链接"面板。

**作用：**
- 显示哪些笔记引用了当前笔记
- 帮你发现知识之间的隐藏联系

**例子：**
你在"数字分身"笔记里，会看到所有提到"数字分身"的地方。

### 4. 全文搜索

**快捷键：**
- Mac：`Cmd + Shift + F`
- Windows：`Ctrl + Shift + F`

**威力：**
- 瞬间搜索所有笔记
- 支持 grep 模式（高级搜索）
- 找到任何你想找的内容

## 第五部分：配置 Git 自动备份（可选但推荐）

### 为什么需要 Git 插件？

- ✅ 自动备份到 GitHub
- ✅ 跨电脑同步（工作时/家里时）
- ✅ 版本管理，随时回滚
- ✅ 免费，不限空间

### 安装 Obsidian Git 插件

1. 打开 Obsidian
2. 点击"设置"（齿轮图标）
3. 点击"第三方插件"
4. 点击"浏览"
5. 搜索"Obsidian Git"
6. 点击"安装"
7. 启用插件

### 配置 Git 插件

**前提：需要先有 GitHub 账号**

1. 创建 GitHub 仓库（免费）
   - 访问：https://github.com/
   - 点击"New"→"New repository"
   - 仓库名：比如"my-notes"
   - 选择"Private"（私有）
   - 点击"Create repository"

2. 在 Obsidian Git 插件中配置：
   - 仓库地址：`你的用户名/仓库名`
   - 分支：`main`
   - 自动提交间隔：30分钟
   - 自动拉取前：先提交本地修改

### 测试同步

1. 写一条测试笔记
2. 等待30秒（自动提交）
3. 打开 GitHub 仓库，看看是否同步成功

**成功后，你的笔记就安全地备份在云端了。**

## 第七部分：常见问题 FAQ

### Q1：数据安全吗？

**A：非常安全。**

1. **数据在本地** - 所有笔记都在你电脑里
2. **GitHub 备份** - 用 Git 插件自动备份
3. **随时导出** - 想换工具随时导出

**说句大实话：** 比存在别人的服务器上安全多了。

### Q2：换电脑怎么办？

**A：超简单。**

**方式1：用 GitHub 同步**
1. 新电脑安装 Obsidian
2. 安装 Git 插件
3. 配置同一个仓库
4. 自动拉取所有笔记

**方式2：手动复制**
1. 把知识库文件夹复制到 U 盘
2. 拷到新电脑
3. 用 Obsidian 打开这个文件夹

### Q3：手机端能用吗？

**A：完全可以。**

1. 下载 Obsidian 手机 app
2. 用 GitHub 同步
3. 或者用 iCloud、Dropbox 同步文件夹

### Q4：笔记太多会很慢吗？

**A：不会。**

Obsidian 本身很轻量：
- 加载速度只和当前笔记有关
- 搜索有索引，很快
- 几万个笔记也没问题

### Q5：需要付费吗？

**A：个人使用完全免费。**

- 基础功能：免费
- 插件：大部分免费
- 同步：用 GitHub 免费

**只有发布官方付费插件才需要授权。**
