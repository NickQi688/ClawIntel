# 流程图 - 选题 Skill 完整流程

Layout: 水平从左到右，四个步骤用箭头连接

STEPS:
1. **数据抓取** - 输入端
   - 动作：自动抓取小红书、抖音热门内容
   - 图标：爬虫/收集器图标
   
2. **方法论拆解** - 处理端
   - 动作：按 SCQA/第一性原理做拆解和扩展
   - 图标：分析/拆解图标
   
3. **选题生成** - 输出端
   - 动作：生成待选选题列表
   - 图标：创意/灯泡图标
   
4. **人工决策** - 终审
   - 动作：用户只需"选或不选"
   - 图标：勾选/人决策图标

CONNECTIONS: 粗箭头从左到右连接，箭头带轻微渐变。前三个步骤是自动化的（浅蓝背景），最后一个是人工环节（浅绿背景）

COLORS:

- 步骤2: 紫色 (#8B5CF6)
- 步骤3: 橙色 (#F59E0B)

- 连接箭头：灰色 (#6B7280)
- 背景：浅灰白 (#F9FAFB)
