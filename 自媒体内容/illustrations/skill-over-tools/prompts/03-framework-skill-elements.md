# 框架 - Skill 三要素

STRUCTURE: 三角形框架，三个角分别放置一个要素，中心是"业务 Skill"核心词

NODES:
1. **Process (流程)** - 右上角
   - 确定性的动作先后顺序
   - 图标：有序的列表/步骤图标

2. **Methodology (方法论)** - 左上角
   - 行业公认的底层逻辑框架
   - 图标：拼图/积木块

3. **Judgment (判断标准)** - 底部中心（最关键）
   - 什么是"好"，什么是"垃圾"
   - 图标：放大镜/准心

CENTER: "业务 Skill" 核心词

COLORS:
- Process: 蓝色 (#3B82F6)
- Methodology: 黄色 (#F59E0B)
- Judgment: 红色 (#EF4444) - 强调其关键性
- 中心核心：深灰 (#374151)
- 连接线：灰色 (#9CA3AF)
- 背景：米白 (#FEFCE8) 或浅奶油色

STYLE: 极简扁平，三角形框架清晰可见。每个要素用圆角矩形或标签框包裹。底部加一行公式：V_skill = 清晰的边界 × 确定的逻辑 × 闭环的反馈
