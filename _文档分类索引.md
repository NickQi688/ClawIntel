# 文档分类索引

这个资料库按“用途 + 主题”整理：

- `学习&AI应用/`：AI 工具学习、提示词、编程工具、Obsidian、商业成长等学习资料。
- `Web 3.0/`：Web3 干货、交易、预测市场、币安内容挖矿等长期主题。
- `羊毛机会/`：AI 赠金、API、区块链和其他可执行机会。
- `项目机会/`：已经形成项目雏形或值得持续跟进的机会。
- `自媒体内容/`：自己创作或准备发布的文章、选题、配图和小红书内容。
- `小红书内容/`：小红书发布流程、文案模板、封面素材和待发布稿件。
- `剪藏内容/`：外部剪藏资料，已按主题拆分为 AI 工具、AI 创作、GEO、Web3、新媒体、商业认知。
- `工作/`：客户项目、交付脚本、视频方案和工作资料。
- `My-Digital-Self/`、`数字分身日记/`、`任务/`：数字分身系统、账号配置、任务和复盘。
- `skills/`：本地可执行 skill 或工具代码；普通文章已移到 `学习&AI应用/编程相关/skill/`。
- `归档/`：空白、过期或暂时不进入主分类的材料。

新文件建议先判断它是“学习资料、机会、项目、自媒体、工作、剪藏、归档”中的哪一种，再放到对应目录。
